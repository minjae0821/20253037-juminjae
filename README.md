# !리눅스 명령어 중에서 top, ps, jobs, kill에 대해 조사해보자!


## 명령어 개요 테이블

| 명령어 | 핵슴 역할 | 대표 키 | 대표 사용 예시 | 권장 사용 시점 | 
|---|---|---|---|---|
| top | 실시간 시스템/프로세스 모니터링 | `P`, `M`, `1`, `k`, `u` | `top` | 즉시 상태 파악/정렬/종료 필요 시 |
| ps | 프로세스 스냅샷 조회 | `aux`, `-ef`, `-o` | `ps aux | grep nginx` | 특정 프로세스 탐색/스크립트용 |
| jobs | 현재 셸의 백그라운드 작업 조회 | (옵션 적음) | `jobs -l` | 포그라운드/백그라운드 전환 관리 |
| kill | 시그널로 프로세스 제어 | `-SIGTERM`, `-SIGKILL`, `-l` | `kill -9 1234` | 안전 종료→강제 종료 순서 권장 |
